## litegpt

This project is a minimal, educational implementation of a GPT-style language model built entirely from scratch, designed to demonstrate the core mechanics behind modern text generation systems. At its heart, the model is trained on a simple Shakespeare corpus, allowing it to learn the statistical structure, rhythm, and vocabulary of Early Modern English. The architecture follows the transformer-based design popularized by GPT models, including token embeddings, multi-head self-attention, and feedforward layers, but implemented in a lightweight and highly readable way using TensorFlow. The goal of this project is not to achieve state-of-the-art performance, but to provide a clear, hands-on understanding of how language models process sequences, learn contextual relationships, and generate text one token at a time. From tokenizer training to sequence batching and autoregressive generation, every component is intentionally kept transparent and hackable.

Once trained, the model is capable of producing Shakespeare-like continuations given any input prompt, mimicking the stylistic patterns it has learned during training. While it does not possess true conversational ability or factual understanding, it demonstrates the fundamental behavior of autoregressive language models: predicting the next token based on prior context. The project includes a simple inference pipeline that allows users to input a prompt and receive generated text, showcasing how sampling techniques like temperature scaling and top-k filtering influence output diversity and coherence. Overall, this MiniGPT serves as a foundational stepping stone for understanding larger language models, offering a complete yet compact pipeline—from raw text to generated output—that can be extended, modified, or scaled depending on the learner’s goals.
